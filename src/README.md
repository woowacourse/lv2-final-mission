# 운동 클래스 예약 서비스

## 도메인 목록

### Member
- [x] 이름, 이메일, 비밀번호에 대한 정보를 가진다.

### Sport
- [ ] 운동 종목, 인원에 대한 정보를 가진다.

### SportSlot
- [ ] Sport와 수업 날짜, 수업 시간에 대한 정보를 가진다.

### Reservation
- [ ] SportSlot과 예약 일시에 대한 정보를 가진다.


## 기능 구현 목록

### 로그인
- [x] 이메일과 비밀번호로 로그인할 수 있다.

### 예약
- [ ] 사용자는 SportSlot을 예약할 수 있다.
  - [ ] 이미 모집 일시가 마감된 SportSlot은 예약할 수 없다.
  - [ ] 이미 모집 인원이 마감된 SportSlot은 예약할 수 없다.
  - [ ] 이미 예약한 SportSlot은 예약할 수 없다.

- [ ] 예약 성공 시 예약자에게 이메일을 전송한다.
  - [ ] SendGrid API를 사용하여 예약자에게 이메일을 전송한다.

### 예약 조회
- [ ] 사용자는 전체 예약 현황을 확인할 수 있다.
  - [ ] 각 SportSlot의 예약 인원을 확인할 수 있다.

- [ ] 사용자는 본인의 예약에 대한 상세한 정보를 확인할 수 있다.
  - [ ] 사용자는 운동 종목, 인원, 수업 날짜, 수업 시간, 예약 일시를 확인할 수 있다.

### 예약 수정
- [ ] 사용자는 본인의 예약을 수정할 수 있다.

### 예약 삭제
- [ ] 사용자는 본인의 예약을 삭제할 수 있다.
