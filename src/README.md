# 운동 클래스 예약 서비스

## 도메인 목록

### Member
- [x] 이름, 이메일, 비밀번호에 대한 정보를 가진다.

### Lecture
- [x] 운동 종목, 수업 날짜에 대한 정보를 가진다.

### Reservation
- [x] 예약 날짜와 Lecture, Member에 대한 정보를 가진다.


## 기능 구현 목록

### 로그인
- [x] 이메일과 비밀번호로 로그인할 수 있다.

### 예약
- [x] 사용자는 Lecture를 예약할 수 있다.
  - [ ] 이미 모집 날짜가 마감된 Lecture는 예약할 수 없다.
  - [ ] 이미 예약한 Lecture는 예약할 수 없다.

- [ ] 예약 성공 시 예약자에게 이메일을 전송한다.
  - [ ] SendGrid API를 사용하여 예약자에게 이메일을 전송한다.

### 예약 조회
- [x] 사용자는 전체 예약 현황을 확인할 수 있다.
  - [x] 각 예약의 예약 번호, 운동 종목, 수업 날짜, 예약자 이름을 확인할 수 있다.

- [x] 사용자는 본인의 예약에 대한 상세한 정보를 확인할 수 있다.
  - [x] 사용자는 예약 번호, 운동 종목, 수업 날짜, 예약 날짜를 확인할 수 있다.

### 예약 수정
- [ ] 사용자는 본인의 예약을 수정할 수 있다.

### 예약 삭제
- [x] 사용자는 본인의 예약을 삭제할 수 있다.
