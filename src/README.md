# 멍구와의 약속 예약하기

## 주요 기능
1. 멍구는 가능한 약속 시간을 등록할 수 있습니다.
2. 사용자는 멍구의 약속시간에 대하여 예약할 수 있습니다.
3. 사용자는 예약 기록을 확인할 수 있습니다.
4. 사용자는 특정 예약 상세 정보를 확인할 수 있습니다.

## 도메인 용어

PlanDate
- 약속 날짜

TimeSlot
- 가능한 시간 슬롯
- 약속의 시작 시간 - 끝나는 시간을 의미

ReservationSlot
- 예약 슬롯
- 약속 날짜와 시간 슬롯으로 하나의 예약 슬롯을 나타낼 수 있음

- Reservation
- 예약
- 예약자와 예약 슬롯으로 하나의 예약을 나타낼 수 있음

## 구현한 기능

admin-reservation
- [ ] 관리자는 예약슬롯을 생성할 수 있다.
- [ ] 관리자는 예약 현황을 모두 조회할 수 있다. 

user-reservation
- [ ] 사용자는 예약을 등록할 수 있다.
- [ ] 사용자는 예약 기록들을 조회할 수 있다.
- [ ] 사용자는 예약 상세정보를 조회할 수 있다.

user
- [ ] 유저는 로그인/로그아웃할 수 있다.
- [ ] 관리자와 사용자인지 Jwt로 검사한다.
- [ ] 관리자만 이용할 수 있는 API는 권한검사를 진행한다.
