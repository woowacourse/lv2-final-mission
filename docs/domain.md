# 도메인 설계

## 맛집 (Entity)

- 맛집 ID
- 맛집 상세 정보
    - 맛집 이름
    - 맛집 주소
    - 맛집 한줄 소개

### 맛집 예약 스케줄 (운영 정보) -> 맛집에 종속적

- 스케줄 ID
- 맛집 (N:1)
  ~~- 예약 (1:N)~~
- 날짜
- 시간
- 예약 가능한 최대 인원

## 예약 (Entity)

- 예약 ID
- 맛집 예약 스케줄 (1:1)
- 예약자
    - 닉네임
    - 이메일
    - 예약 비밀번호
- 예약 상세 정보
    - 예약 인원
    - 예약 메시지
