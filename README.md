# 회의실 예약 서비스 기능 구현 목록

## 예약 API
### 예약 생성
- [ ] 사용자는 특정 회의실을 예약할 수 있다.

### 예약 조회
- [ ] 모든 사용자는 예약 현황을 확인할 수 있다.
- [ ] 사용자 본인은 자신이 한 예약의 상세 정보까지 확인할 수 있다.

### 예약 수정
- [ ] 사용자는 본인의 예약만 수정하고 삭제할 수 있다.

## 회의실 API
### 회의실 조회
- [ ] 모든 회의실 목록을 조회할 수 있다.

## 이메일 전송 API
- [ ] SendGrid API를 연동해 예약 결과에 대한 이메일을 사용자에게 발송한다.
