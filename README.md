# 지하철 좌석 예약 서비스
2025년 6월 서울시에서 지하철 좌석 예약 시스템을 일부 차량에 한정하여 시범 운영합니다.  
현재 시범 대상 열차는 2호선 차량의 1-1칸 5개 좌석입니다.
즉, 현재는 2호선의 1-1칸만 좌석 예약이 가능합니다.

[주요 기능]
1. 사용자는 지하철의 특정 좌석을 예약할 수 있다.
2. 모든 사용자는 예약 현황을 확인할 수 있다.
3. 사용자는 본인의 예약 현황을 상세하게 알 수 있다.
4. 사용자는 본인의 예약을 수정/삭제할 수 있다.

[도메인 지식]
1. 각 열차엔 열차를 특정하는 번호가 있습니다. 현재는 1번부터 9번까지 있다고 가정합니다.
2. 열차의 1-1칸만 좌석 예약 서비스를 시범 운행 중이며, 1-1칸엔 총 5석의 좌석이 있습니다.
3. 각 좌석은 A, B, C, D, E번 좌석으로 구분됩니다.
4. 2호선은 순환선입니다. 승객은 항상 최단거리 방향으로 이동하는 열차를 탄다고 가정합니다.

[//]: # (4. 열차 운행은 오전 6시부터 ~ 자정&#40;0시&#41;입니다.)
