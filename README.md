# 팝업 스토어 예약 서비스

## 기능 목록

### 팝업 스토어 기능

- 오픈하기
    - 점주는 팝업 스토어를 오픈할 수 있다.
        - 팝업 스토어를 저장한다.

- 팝업 스토어 도메인 모델
    - id: 팝업 ID
    - title: 팝업 제목
    - content: 팝업 설명
    - startAt: 시작일
    - endAt: 종료일
    - maxEnteredMemberCount: 최대 수용 인원
    - shopkeeper: 상점주

### 예약 기능

- 예약 하기
    - 사용자가 예약한다.
        - 예약을 저장한다.
            - 기준 인원 이상이 대기 중이면 대기 상태로 저장한다.
            - 기준 인원 미만이 대기 중이면 참여 중 상태로 저장한다.

- 순번 조회
    - 같은 팝업 스토어에 대해 나의 예약이 몇 번째 대기인지 확인할 수 있다.

- 나가기
    - 팝업 스토어에서 나간다.
        - 나가면 내 예약은 LEAVE 상태로 변경된다.
        - 내 바로 뒤에 대기자는 ENTERED 상태로 변경된다.

- 예약 도메인 모델
    - id: 예약 ID
    - member: 예약자
    - popupStore: 예약할 상점
    - reservedAt: 예약 시간

### 멤버 기능

- 가입할 수 있다.

- 멤버 도메인 모델
    - id: 멤버 ID
    - name: 이름
    - nickname: 닉네임

### 점주 기능

- 가입할 수 있다.

- 점주 도메인 모델
    - id: 점주 ID
    - name: 이름
