# 회의실 예약 서비스

### **일반 회원 기능**

#### 회원 가입

- 서비스 사용자는 회원 가입을 할 수 있다.
- 회원 가입을 위해 `아이디`, `이메일`, `패스워드`를 입력해야 한다.
- 이미 사용 중인 `아이디`, `이메일`일 경우, 예외 메시지를 응답한다.

### **관리자 회원 기능**

- 관리자는 예약 정보를 관리할 수 있다.
- 관리자가 아닌 사용자가 접근할 경우, 예외 메시지를 응답한다.

#### 회의실 장소 추가 기능

- 관리자는 회의실 장소를 추가할 수 있다.
- 회의실 장소 등록을 위해 `회의실 이름`, `회의실 설명`, `가용 인원`을 입력해야 한다.
- 이미 존재하는 `회의실 이름`을 입력한 경우, 예외 메시지를 응답한다.
- 유효하지 않은 `가용 인원`을 입력한 경우, 예외 메시지를 응답한다.
    - `가용 인원` <= 0

#### 회원 예약 블랙리스트 관리 기능

- 관리자는 회원의 예약 블랙리스트를 관리할 수 있다.
- 블랙리스트에 존재하는 회원은 예약 시도 시, 예외 메시지를 응답한다.
- 예약 블랙리스트 등록을 위해 `사용자 ID`, `블랙리스트 사유`를 입력해야 한다.
- 이미 블랙리스트에 존재하는 `사용자 ID`를 입력한 경우, 예외 메시지를 응답한다.
- `블랙리스트 사유`를 입력하지 않은 경우, 예외 메시지를 응답한다.


- 예약 블랙리스트 취소를 위해 `사용자 ID`, `블랙리스트 취소 사유`를 입력해야 한다.
- 블랙리스트에 존재하지 않는 `사용자 ID`를 입력한 경우, 예외 메시지를 응답한다.
- `블랙리스트 사유`를 입력하지 않은 경우, 예외 메시지를 응답한다.

### **회원 공통 기능**

#### 로그인/로그아웃

- 서비스 사용자는 로그인/로그아웃을 할 수 있다.
- 로그인을 위해 `아이디`, `패스워드`를 입력해야 한다.
- 존재하지 않는 `아이디`이거나 유효하지 않은 `패스워드`를 입력한 경우, 예외 메시지를 응답한다.

#### 예약 기능

- 사용자는 `회의실 장소`, `날짜`, `시간`을 선택하여 예약할 수 있다.
- 휴일 및 공휴일은 예약할 수 없다.
- 이미 존재하는 `날짜`, `시간`, `회의실 장소` 예약의 경우, 예외 메시지를 응답한다.

#### 예약 조회 기능

- 사용자는 예약 현황을 조회할 수 있다.

#### 예약 수정/삭제 기능

- 사용자는 본인의 예약만 수정/삭제할 수 있다.
- 본인의 예약이 아닌 경우, 예외 메시지를 응답한다.
