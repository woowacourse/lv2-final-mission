# 영화관 예약 서비스

영화관 관리자, 이용자의 영화 예매를 도와주는 서비스입니다.

## 관리자 서비스

- 영화 정보 등록 기능
- 영화 슬롯 추가 기능

# 비회원 서비스

- 상영 영화 목록 조회 기능

# 회원 서비스

- 회원 가입
- 영화 예약 기능
- 예약한 영화 조회 기능
- 예약한 영화 취소 기능

# 메모

## 기능 요구사항

- 회원 기능
- 모든 사용자는 예약 현황 확인 가능
- 사용자 본인의 예약 상세 정보 확인 가능
- 사용자 본인의 예약만 수정 및 삭제 가능
- 구현한 기능은 문서화 하여 작성.

## 프로그래밍 요구사항 

- 데이터베이스를 사용하여 데이터를 관리
- 필수 기능을 테스트 코드를 통해 검증.
- 외부 API 연동 필수
    - 공휴일 정보를 조회하여 공휴일에 영화 가격을 1.5배 부과.
