# 회의실 예약

## 요구 사항

- [x]  사용자는 이메일, 비밀번호로 회원가입한다
- [x] 사용자의 이름은 랜덤으로 생성한다
- [x]  사용자는 이메일과 비밀번호로 로그인한다
- [x]  초기에 회의실이 등록되어있다
- [x] 회의실은 이름을 가진다
- [x] 회의실은 최대 이용 시간을 가진다

- [x]  회의실에는 예약을 할 수 있다
    - [x]  예약은 회의실의 최대 예약 시간을 초과하여 할 수 없다
    - [ ]  이미 예약중인 시간에는 예약할 수 없다
    - [x]  과거 시간에는 예약할 수 없다
    
- [x]  사용자는 자신의 예약을 확인할 수 있다
- [x]  모든 사용자는 회의실 별 예약을 확인할 수 있다
- [x]  사용자는 자신의 예약의 상세 정보를 확인할 수 있다
- [x]  사용자는 본인의 예약을 수정하고 삭제할 수 있다
