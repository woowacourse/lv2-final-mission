## 크루 닉네임 예약 서비스
### 우아한테크코스 내 크루들의 닉네임을 관리한다.

### 기능 목록
- [x] 닉네임을 예약할 수 있다.
  - [x] 닉네임의 길이는 2-4글자이다.
  - [x] 한 크루 당 최대 2개까지 예약할 수 있다.
  - [x] 닉네임은 중복될 수 없다.
  - [x] 이미 확정한 닉네임이 있을 경우 추가로 예약할 수 없다.


- [x] 예약 중인 닉네임 목록을 조회할 수 있다.


- [x] 자신의 예약 목록을 조회할 수 있다.


- [x] 닉네임 예약을 취소할 수 있다.
  - [x] 자신의 예약만 취소할 수 있다.
  - [x] 한 번 확정된 예약은 취소할 수 없다.


- [x] 원하는 길이(2-4글자)의 닉네임을 추천받을 수 있다.


- [x] 예약한 닉네임을 확정할 수 있다.
  - [x] 다른 닉네임 예약은 자동으로 취소된다.


- [x] 예약한 닉네임을 수정할 수 있다.
  - [x] 자신의 예약만 수정할 수 있다.
  - [x] 수정 전 닉네임과 동일할 수 없다.
  - [x] 다른 예약 및 확정 닉네임과 중복될 수 없다.
