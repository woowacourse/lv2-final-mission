# 강의실 좌석 예약 시스템

## 기능 요구 사항

### 예약 관련

- 사용자는 강의실의 좌석을 예약할 수 있다.
- 사용자는 자신의 예약 현황을 확인할 수 있다.
- 사용자는 자신의 예약 상세 정보를 확인할 수 있다.
- 사용자는 예약을 수정할 수 있다.
- 사용자는 예약을 삭제할 수 있다.

### 인증 / 인가 관련

- [x] 사용자는 회원가입을 할 수 있다.
- [x] 사용자는 로그인을 할 수 있다.
- [ ] 사용자는 예약이 완료되면 이를 이메일로 받아볼 수 있다.

## 정책

### 예약 관련

- 사용자는 현재 시간 이전에 대한 예약을 할 수 없다.
  ex) 2025년 6월 9일 15시 01분에는 2025년 6월 9일 15시 02분에 대한 예약부터 가능
