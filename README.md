## 크루 닉네임 예약 서비스
### 우아한테크코스 내 크루들의 닉네임을 관리한다.

### 기능 목록
- [x] 닉네임을 예약할 수 있다.
  - [x] 닉네임의 길이는 2-4글자이다.
  - [x] 한 크루 당 최대 2개까지 예약할 수 있다.
  - [x] 닉네임은 중복될 수 없다.
- [ ] 모든 예약 중인 닉네임을 확인할 수 있다.
- [ ] 예약 중인 닉네임 목록을 조회할 수 있다.
- [ ] 자신의 예약 목록을 조회할 수 있다.
- [x] 예약 정보를 삭제할 수 있다.
  - [x] 자신의 예약만 삭제할 수 있다.
- [ ] 원하는 길이(2-4글자)의 닉네임을 추천받을 수 있다.
