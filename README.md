# 운동 클래스 예약 서비스

## 기능

- [x] 이메일, 패스워드를 입력하면 회원가입 가능합니다.
    - [x] 회원가입 시 이름은 자동으로 생성된 랜덤 이름을 사용합니다.
    - https://randommer.io/ 의 랜덤 이름 생성 API를 사용합니다.
- [x] 사용자는 운동 클래스를 예약할 수 있습니다.
    - 사용자는 수업, 날짜, 시간을 선택하여 수업을 예약할 수 있습니다.
    - `POST /reservations` 요청 시 해당 정보를 요청 바디에 담아 보내야 합니다.
    - [x] 로그인 된 사용자 정보를 받아옵니다.
- [ ] 모든 사용자는 예약 현황을 확인할 수 있습니다.
- [x] 사용자 본인은 자신이 한 예약의 상세 정보까지 확인할 수 있습니다.
    - `GET /reservations/mine` 요청 시 사용자의 모든 예약 정보를 조회할 수 있습니다.
- [x] 사용자는 본인의 예약만 삭제할 수 있습니다.
    - `DELETE /reservations/{id}` 요청 시 본인의 예약을 삭제할 수 있습니다.
