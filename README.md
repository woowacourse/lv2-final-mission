## 롤 5:5 내전 구인 및 참가 예약 서비스

### 유저 관련

- [x] 회원가입을 한다.
    - [x] 존재하지 않는 롤 닉네임 및 태그라면 404를 응답한다.
        - RIOT API를 기반으로 검증한다.
    - [x] 이미 가입한 유저라면 400을 응답한다.


- [x] 로그인을 한다.
    - [x] 존재하지 않는 유저라면 404를 응답한다.
    - [x] 비밀번호가 틀리다면 401을 응답한다.


### 내전방 관련

- [x] 유저를 방장으로 하는 내전방을 생성한다.
    - [x] 존재하지 않는 유저라면 404를 응답한다.
    - [x] 내전 시작 날짜 및 시간이 과거라면 400을 응답한다.


- [x] 롤 내전방을 모두 조회한다.


- [x] 롤 내전방 정보와 참여 인원들을 조회한다.
    - [x] 존재하지 않는 내전방이라면 404를 응답한다.


- [x] 해당 내전을 예약한다.
    - [x] 존재하지 않는 내전방이라면 404를 응답한다.
    - [x] 참여자 인원수가 10명이라 꽉 찼다면 400을 응답한다.
    - [x] 이미 해당 내전을 예약한 유저라면 400을 응답한다.
    - [x] 내전 시작 날짜 및 시간을 지났다면 400을 응답한다.


- [x] 유저가 참여중인 내전방을 조회한다.
    - [x] 존재하지 않는 유저라면 404를 응답한다.


- [x] 내전방 참여 예약을 취소한다.
    - [x] 존재하지 않는 유저라면 404를 응답한다.
    - [x] 존재하지 않는 내전방이라면 404를 응답한다.
    - [x] 내전방 참가자가 아닌 유저가 내전방 참여 예약을 취소하면 400을 응답한다.
    - [x] 내전 시작 날짜 및 시간을 지났다면 400을 응답한다.


- [ ] 내전방을 삭제한다.
    - [ ] 내전방 삭제를 하는 유저가 방장이 아니라면 401을 응답한다.


- [ ] 내전방의 정보를 수정한다.
    - [ ] 내전방 수정을 하는 유저가 방장이 아니라면 401을 응답한다.
    - [ ] 수정된 내전 시작 날짜 및 시간이 과거라면 400을 응답한다.
