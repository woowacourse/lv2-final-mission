# 🏟️야구장 좌석 예약 서비스

## 서비스에서 제공하는 api

- [x] 회원가입(/signup POST)
- [x] 로그인(/login POST)
- [ ] 좌석 예약(/reservations POST)
- [ ] 모든 예약 확인(/reservations GET)

## 기능 요구사항

### 회원

- [x] 회원에 대한 이메일, 닉네임, 연령대를 저장한다.
- [x] 할인이 적용되는 나이대인지 확인할 수 있다.

### 좌석 예약

- [ ] 사용자는 본인의 예약만 수정하고 삭제할 수 있다.
- [ ] 사용자는 본인이 앉을 야구장 좌석을 예약할 수 있다.
    - 예약에 필요한 정보는 경기 날짜, 시간, 좌석 등급, 좌석 위치 정보, 회원 정보이다.
- [ ] 평일(월, 화, 수, 목)의 경우 각 좌석별로 아래의 가격이 적용된다.
    - 테이블석 55,000
    - 응원지정석 20,000
    - 3층 지정석 14,000
- [ ] 주말(금, 토, 일) 및 공휴일의 경우 각 좌석별로 아래의 가격이 적용된다.
    - 테이블석 88,000
    - 응원지정석 28,000
    - 3층 지정석 21,000
- [ ] 모든 사용자는 예약 현황을 확인할 수 있다.
- [ ] 사용자 본인은 자신이 한 예약의 상세 정보를 확인할 수 있다.
- [ ] 만 19세 이하일 경우와 만 60세 이상일 경우, 가격의 30%를 할인한다.

### 외부 api 연동

- [ ] 예약 등록 과정에서 공공 데이터포털의 날씨 정보 조회 api를 통해, 비 예보가 있을 경우 우천 취소에 대한 경고를 한다.
- [ ] 가격 적용 과정에서 공공 데이터포털의 공휴일 정보 조회 api를 통해 공휴일 여부를 확인한다.
