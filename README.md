## 맛집 예약 서비스

### 회원

- [x] 회원가입, 로그인
  - [x] 고객 / 업주로 권한이 분류된다.
- [x] 권한이 필요한 서비스 검증

### 예약 - 고객 회원

- [x] (필수) 예약 내역 확인
- [x] (필수) 예약 생성
  - [x] 존재하지 않는 예약 정보(= 업주가 등록)로는 예약 생성 불가능
  - [ ] (선택) 해당 일자/시간에 예약 가능한 최대 건수를 넘어가는 경우에는 생성이 불가능하다.
- [x] (필수) 예약 내역 상세정보 조회
- [x] (필수) 예약 수정
  - [x] 자신이 생성한 예약에 한해서 수정이 가능하다.
- [x] (필수) 예약 삭제
  - [x] 자신이 생성한 예약에 한해서 삭제가 가능하다.

### 예약 - 업주 회원

- [x] 예약 가능 날짜/시간 업로드
  - [x] 식당, 날짜, 시간에 대해서 중복되는 데이터를 생성할 수 없다. 
  - [x] 존재하지 않는 식당 정보가 입력되는 경우 예외처리

### 식당

- [ ] (선택) 식당 데이터 생성
  - [ ] Owner 권한을 가진 회원만 생성할 수 있다.
- [ ] (선택) 전체 식당 조회

### 외부 API

- [ ] (필수) 예약이 성공적으로 생성되면 예약 완료 메일이 발송된다.
