# 🌸 포스티의 회의실 예약 어플리케이션 🌼

## 기능 목록

- [x] 사용자 회원가입을 할 수 있습니다.
- [x] 사용자 회원탈퇴를 할 수 있습니다.
- [x] 사용자 로그인을 할 수 있습니다.
- [x] 사용자는 특정 회의실을 예약할 수 있습니다.
  - 과거의 시간에 예약이 불가능합니다.
  - 종료 시간이 시작 시간보다 빠를 수 없습니다.
  - 10시 ~ 22시까지 예약 가능합니다.
- [x] 모든 사용자는 회의실 예약 현황을 확인할 수 있습니다.
- [x] 모든 사용자는 회의실과 날짜를 지정해 회의실 예약 현황을 확인할 수 있습니다. 
- [x] 사용자 본인은 자신이 한 예약의 상세 정보까지 확인할 수 있습니다.
- [x] 사용자는 본인의 예약만 수정하고 삭제할 수 있습니다.
  - 예약 수정은 시간만 가능합니다
- [x] 예약이 성공하면 사용자에게 메일을 전송합니다.
- [x] 회의실을 생성할 수 있습니다.
  - 중복된 이름의 회의실은 생성할 수 없습니다.
- [x] 전체 회의실을 조회할 수 있습니다.
- [x] 회의실을 삭제할 수 있습니다.
