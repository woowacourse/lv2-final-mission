# 우아한 테크코스 최종 미션 - 예약 서비스

## 📌 기능 요구사항

- [x] 사용자는 특정 대상을 예약할 수 있습니다.
- [x] 모든 사용자는 예약 현황을 확인할 수 있습니다.
- [x] 사용자 본인은 자신이 한 예약의 상세 정보까지 확인할 수 있습니다.
- [x] 사용자는 본인의 예약만 수정하고 삭제할 수 있습니다.

## 📌 프로그래밍 요구사항

- [x] 데이터베이스: H2 또는 MySQL을 활용하여 데이터를 관리합니다.
- [] 테스트: 애플리케이션이 의도한 대로 잘 동작하는지 테스트 코드를 통해 검증해야 합니다.
- [x] 외부 API 연동 : 외부 API를 최소 1개 이상 연동하여 서비스에 활용해야 합니다.

# 목표한 예약 서비스: 뷰티 샵 예약 서비스

## 구현 기능

- [x] 사용자는 뷰티 샵의 예약을 할 수 있습니다.
- [x] 예약을 하기 위해서는 날짜, 시간, 서비스 종류, 담당자를 선택해야 합니다.
- [x] 사용자는 본인이 예약한 예약을 확인할 수 있습니다.
- [x] 사용자는 본인의 예약을 삭제할 수 있습니다.

### 예약 생성의 조건

#### 날짜 조건

- [x] 공휴일은 예약할 수 없습니다.
    - 공휴일은 외부 API를 통해 확인합니다.

#### 시간 조건

- [] 뷰티 샵은 10시부터 20시까지 운영합니다. 즉, 10시에서 20시 안에 서비스가 끝나는 경우에만 예약이 가능합니다.
    - 만약 소요 시간을 더하면 20시가 넘는 경우 예약할 수 없습니다.

- [] 예약은 현재 시간을 기준으로 1시간 이후부터 가능합니다.

#### 서비스 종류 조건

- [] 선택한 서비스를 제공할 수 있는 담당자가 해당 시간에 존재하지 않는 경우 예약할 수 없습니다.

#### 담당자 조건

- [] 선택한 담당자가 해당 시간에 근무 일정이 있는 경우, 예약할 수 없습니다.
- [] 선택한 담당자가 휴무일인 경우 예약할 수 없습니다.
