# Lv2 최종 미션 repository

## 기능 목록

### Member

- 회원 가입 API
    - 회원 정보를 DB에 저장한다.
- 닉네임 추천
    - 외부 API를 통해 무작위로 닉네임을 추천한다.

### Auth

- 로그인 API
    - 로그인에 성공하면 JWT 토큰을 발급한다.

### Reservation

- 예약 생성 API
    - 화장실 예약을 생성한다.
- 예약 현황 조회 API
    - 선택한 날짜와 화장실의 예약을 조회한다.
- 내 예약 조회 API
- 예약 삭제 API

### Toilet

- 화장실 목록 조회
    - 전체 화장실 목록을 조회한다
